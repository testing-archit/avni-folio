## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

- Update portfolio items in the `portfolioItems` array in `App.jsx`
- Modify colors and styling in Tailwind classes
- Adjust 3D scene parameters in the `HeroScene` component
- Update contact information in the contact section
